# How to Install Icarus Verilog

Installing Icarus Verilog (iverilog) varies by operating system. For a complete simulation environment, it is highly recommended to install GTKWave alongside it to view simulation waveforms.

## 1. Windows

- Download the Icarus Verilog installer from the [official website](https://bleyer.org/icarus/)
- Run the installer and follow the on-screen instructions to complete the installation
- To install GTKWave, download it from its official website and run the installer similarly

## 2. macOS

- If you have Homebrew installed, you can install Icarus Verilog by running:
  ```bash
  brew install icarus-verilog
  ```
- To install GTKWave, use:
  ```bash
  brew install gtkwave
  ```

## 3. Linux (Ubuntu/Debian)

- Open a terminal and run the following command to install Icarus Verilog:
  ```bash
  sudo apt-get install iverilog
  ```
- To install GTKWave, use:
  ```bash
  sudo apt-get install gtkwave
  ```
